Privacy Policy

InfiniTD is an open-source web game provided at no cost and is intended for use as is. This policy describes how the game collects and uses user information. Your information will not be shared with anyone except as described in this Privacy policy.

What We Collect and How We Use It

InfiniTD collects three kinds of data: login data, game data, and logs.

Login data is the minimal necessary data to allow users to login through our third-party authentication providers. This is handled by the third party Firebase Authentication API and may include an identifier such as an email address. This data is used solely for authentication and will not be shared unless legally required.

Game data is the data generated from interacting with the game such as username, tower placement, and gold. It is connected to the login data only through a pseudononymous user identifier. This data is necessarily exposed publicly by the game for normal functioning.

Logs include both the game backend logs and infrastructure logs.

The game backend logs consist of game data and data derived from interaction with the game. This is used to improve the game (such as identifying and fixing bugs). Game backend logs are currently publicly exposed.

Infrastructure logs include the system logs of the machine the game is running on as well as the logs of HTTP server and logs from other necessary software for the game’s operation. This data may include your device’s Internet Protocol (IP) address, browser version, operating system, and time of use. These logs are only used for improving the game and will not be share unless legally required.

Data Deletion

All login and game data may be deleted by deleting your account from the account page. This will not delete any data stored in the logs.

Changes to this Privacy Policy

We may update our Privacy Policy from time to time. Thus, you are advised to review this page periodically for any changes. We will notify you of any changes by posting the new Privacy Policy on this page. These changes are effective immediately, after they are posted on this page.
